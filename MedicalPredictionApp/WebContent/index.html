Smart Health Care

A Smart Approach To Health Care Technology Using Big Data Analysis

About Smart Health Care

Smart Health Care is an application with an approach towards the problem of gathering clinical data from smaller geographic regions and perform big data analysis to determine health patterns, statistics. This includes analyzing and processing large volumes of data using NoSQL databases like MongoDB, advanced data processing techniques like Apache Hadoop, Map-Reduce etc, and analysis tools like R programming language.

The outcome is a useful and structured data pattern for clinical data in the field of medical research. Major benefits/impacts of this project will be for the society where doctors and medical practitioners will have more comprehensive and structured information about diseases, symptoms, causes, and patient's health. It will help the people in a region survive from epidemic diseases and aware them on better ways to improve nutrition and hygiene. This type of analysis can help the economists and governments to streamline the expenditure and address the most demanding health issue.
